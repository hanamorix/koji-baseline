Diffing Strategy

Row-Level Diff
Each row gets a content hash (fast string concat of char+fg+bg+flags per cell). On new snapshot, compare hash. Changed row → rebuild innerHTML. Unchanged → skip entirely.
Typical frame: 2-5 rows change
Cell-Level Cursor
Cursor position tracked separately. On move: remove class from old cell, add to new cell. Two DOM operations max. Cursor blink via CSS animation — zero JS.
60fps blink with 0 RAF cost
Frame Batching
During heavy PTY output, multiple snapshots can arrive per frame. Only render the latest snapshot per rAF. Drop intermediate frames — user only sees the result.
Prevents DOM thrash on `cat bigfile`

Font Fallback Chain

CSS font-family resolution order
1. Selected font (e.g. 'JetBrains Mono') — ligatures, Latin, symbols
2. 'Apple Color Emoji' — emoji rendering (macOS native)
3. 'Hiragino Sans', 'Noto Sans CJK SC' — Japanese/Chinese/Korean
4. monospace — system fallback

Font Controls

/font
Interactive picker menu (same style as /theme). Shows font name + live preview line.
Cmd+/Cmd-
Increase/decrease font size by 1px. Range: 10px–24px. Grid recalculates rows/cols.
Ligature Toggle
Setting in config. CSS font-variant-ligatures: normal | none. Default: on.
