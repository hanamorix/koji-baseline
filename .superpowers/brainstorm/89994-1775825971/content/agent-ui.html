A. Inline Agent

Agent messages appear inline in the terminal flow with a left border. Commands show in a code block with approve/reject controls. "AGENT" badge in top bar, "agent active" in bottom bar. Your terminal stays usable ‚ÄĒ agent and shell share the space.

B. Split Pane Agent

Agent gets its own pane ‚ÄĒ terminal on the left, agent conversation on the right. Both always visible. You can type in either pane. More space for the agent to show results without blocking your terminal.

C. Full-Screen Agent Mode

Agent takes over the entire terminal viewport ‚ÄĒ like Claude Code does. Dedicated chat interface with your prompt at the bottom. Command output shows inline. Press Esc to return to normal terminal. Ctrl+T toggles between agent and terminal.
