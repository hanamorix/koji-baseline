Smooth Scrolling

Trackpad
Native CSS scroll-behavior: smooth on the grid-scroll container. macOS momentum scrolling works automatically. No JS scroll handler needed.
Keyboard
Shift+PageUp/Down — scroll by viewport height
Shift+↑/↓ — scroll by single line
Shift+Home/End — jump to top/bottom of scrollback
Auto-scroll
New output auto-scrolls to bottom unless the user has scrolled up. Once scrolled up, output continues below without yanking viewport. Scrolling to bottom re-enables auto-scroll.

Selection & Clipboard

Selection (Mouse)
• Click + drag — select text range
• Double-click — select word
• Triple-click — select line
• Shift+click — extend selection
• Selection highlight: theme accent at 25% opacity
DOM grid makes this near-free. Browser handles all selection mechanics. We style ::selection to match theme.
Clipboard Operations
• Cmd+C — copy selection (if any), else send SIGINT
• Copy-on-select — auto-copy when mouse released (toggleable)
• Cmd+V — paste from clipboard → PTY (bracketed)
• Middle-click — paste from clipboard → PTY
• Bracketed paste — wraps pasted text in \e[200~...\e[201~
Cmd+C smart behaviour: if text is selected, copy it. If nothing selected, send ^C (SIGINT) to PTY. Best of both worlds.

Unified Animation Loop

v0.3 — Three RAF Loops
1. Cursor blink (always running)
2. Grid render (on every PTY event)
3. Idle scan line + kanji (after 30s)
All three fighting for frame budget
v0.4 — Minimal Approach
• Cursor blink → CSS animation (zero JS)
• Grid render → event-driven (on PTY snapshot only)
• Kanji cycle → setInterval(800ms) (no RAF needed)
No RAF loops running at idle
