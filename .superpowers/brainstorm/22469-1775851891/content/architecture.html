Kōji v0.4 — Terminal Quality Pivot

Architecture overview — does this capture what we agreed on?

Core Change: Canvas → DOM Grid

Rendering Pipeline (Before → After)
❌ v0.3 (Canvas)
PTY bytes
  ↓
alacritty_terminal (VTE parse)
  ↓
GridSnapshot JSON (all 1920 cells)
  ↓
Canvas fillRect + fillText (full redraw)
  ↓
No selection, no ligatures, no emoji
✓ v0.4 (DOM Grid)
PTY bytes
  ↓
alacritty_terminal (VTE parse)
  ↓
GridSnapshot JSON (all cells)
  ↓
Diff against shadow grid
  ↓
Row-level DOM updates (changed rows only)
Cell-level cursor update (2 cells max)
  ↓
Native ligatures, emoji, selection ✓

What Gets Removed

Waveform Animation
Both top and bottom strips. CPU data stays in status bar.
Canvas Scan Line
Idle scan line effect. Kanji cycling stays.
Canvas Grid Renderer
Entire grid.ts Canvas pipeline replaced by DOM.
Scrollback Fade Effect
Canvas-specific dimming. Could revisit as CSS opacity if wanted.

What Gets Added

DOM Grid with Diffing
Row-level content diff, cell-level cursor. Pre-created DOM nodes.
Native Selection + Clipboard
Click-drag, copy-on-select, Cmd+C, middle-click paste, bracketed paste.
Smooth Scrollback (10k default)
Momentum scrolling, fade-on-idle scrollbar, adaptive buffer.
Font System
Curated font picker, ligature toggle, Cmd+/- sizing, emoji+CJK fallback.
TrueColor + Wide Chars
24-bit color fully exercised. Double-width cells for CJK/emoji.
Single RAF Loop
Merge 3 separate loops into 1. Cursor blink + kanji cycle unified.

Decision Summary

Rendering DOM grid (spans in row divs), row-level + cell-cursor diffing
Fonts Curated picker (3-4 on-brand fonts), ligature toggle, Cmd+/- sizing
Scrollback Adaptive, 10k default, configurable, smooth scroll + fade scrollbar
Clipboard Copy-on-select (toggleable) + Cmd+C + middle-click paste + bracketed paste
Waveform Removed. Terminal expands to fill space.
Idle anim Kanji cycling stays. Scan line removed.
Resize Dynamic rows/cols from viewport. Font size picker + Cmd+/-.
